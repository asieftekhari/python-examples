# python-examples
**In this part, I have provided the following examples for you in Python.**

**Example 1:** Design a calculator that takes two numbers as input and performs the following operations: +, -, *, /, ** (Exponentiation Operator), and // (floor Division Operator).

**Example 2:** Write a program that continuously takes input from the user. The program should keep asking for numbers until the user enters -1, at which point the loop terminates and the program ends.

**Example 3:** Write a program that takes a number as input and determines whether it is even or odd.

**Example 4:** Write a program that takes a number as input representing the user's age and prints their age group:

- If greater than 0 years and less than 6 years, output: `"Child"`
- If greater than or equal to 6 years and less than 10 years, output: `"Young Child"`
- If greater than or equal to 10 years and less than 14 years, output: `"Pre-teen"`
- If greater than or equal to 14 years and less than 24 years, output: `"Youth"`
- If greater than or equal to 24 years and less than 40 years, output: `"Adult"`
- If greater than or equal to 40 years, output: `"Middle-aged"`

**Example 5:** Write a program that accepts an operator (such as +, -, *, or /) and two numbers (operands) as input. Then, based on the operator provided, it should perform the appropriate mathematical operation and display the result.

**Example 6:** Write a program that reads a three-digit number, reverses its digits, multiplies the reversed number by 2, and prints the result. You may assume the input is always a three-digit number. Example: Input 765 → Reverse is 567 → 2 × 567 = 1134 (output).

**Example 7:** Write a program that reads an integer and prints the smallest multiple of 10 that is strictly greater than the input number. For instance, input 11 gives 20, and input 40 gives 50.
