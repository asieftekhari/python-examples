# python-examples
**In this part, I have provided the following examples for you in Python.**

**Example1:** Design a calculator that takes two numbers as input and performs the following operations: +, -, *, /, ** (Exponentiation Operator), and // (floor Division Operator).

**Example2:** Write a program that continuously takes input from the user. The program should keep asking for numbers until the user enters -1, at which point the loop terminates and the program ends.

**Example3:** Write a program that takes a number as input and determines whether it is even or odd.

**Example4:** Write a program that takes a number as input representing the user's age and prints their age group:

- If greater than 0 years and less than 6 years, output: `"Child"`
- If greater than or equal to 6 years and less than 10 years, output: `"Young Child"`
- If greater than or equal to 10 years and less than 14 years, output: `"Pre-teen"`
- If greater than or equal to 14 years and less than 24 years, output: `"Youth"`
- If greater than or equal to 24 years and less than 40 years, output: `"Adult"`
- If greater than or equal to 40 years, output: `"Middle-aged"`
